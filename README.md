# android_device_lenovo_kunlun2
For building SHPR for Lenovo Z6 Lite

SHRP device tree for Lenovo Z6 Lite

## Features

Works:

- ADB
- Decryption of /data
- Screen brightness settings
- Correct screenshot color
- MTP
- Flashing (opengapps, roms, images and so on)
- Backup/Restore
- USB OTG
- Double-tap-to-wake
- Vibration
